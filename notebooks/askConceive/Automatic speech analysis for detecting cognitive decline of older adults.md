# 基于自动语音分析检测老年人认知衰退


## 提取了声学特征（基频、抖动、闪烁、MFCC、共振峰）
1) Pitches（基频，Pitch） 
- 含义：声音的基本频率，通常用来描述声带振动的快慢。人声中，基频与音高直接相关。
- 应用：语音情感识别、说话人识别、语音合成、音乐音高估计。
- 常见指标：F0 的平均值、最大/最小 F0、F0 的方差、F0 轮廓的稳定性等。
- 测量方法：自相关、周期性分析、基于短时傅里叶变换的谱分析，或现代方法如声音端点检测。

2) Jitter（呈现为基频不规则性）
- 含义：相邻周期的时长（周期长度）的微小随机波动，反映声带振动的稳定性。
- 应用：语音病理学（如某些声带疾病会提高 jitter）、高质量语音合成需要低 jitter。
- 常见指标：Local jitter、Relative Average Perturbation (RAP)、平均周期长度的变动等。
- 注意：Jitter 过高可能表示信号噪声或病理状态，需要结合其他特征一起分析。

3) Shimmer（振幅不规则性）
- 含义：相邻周期的振幅（能量）波动，反映声带振动幅度的一致性。
- 应用：与 jitter 类似，用于评估声带功能、音质稳定性，以及病理状态指示。
- 常见指标：Local shimmer、Shimmer in dB、S-number 等。
- 注意：Shimmer 常与 jitter 一起使用，以全面评估声门稳定性。

4) MFCCs（Mel-Frequency Cepstral Coefficients，梅尔倒谱系数）
- 含义：代表信号在梅尔尺度上的短时能量分布，是描述音频频谱形状的特征向量。梅尔尺度模仿人耳感知音高的非线性特性。
- 应用：语音识别、说话人识别、音源分离、音乐信息检索等。
- 常见做法：对每一帧音频计算一组 MFCC，通常取前 12-13 个系数（有时再加上能量项和一阶/二阶差分系数，以捕获动态信息）。
- 解释：前几项 MFCC 抽取了主频谱形状信息，较高的系数往往包含噪声信息，因此通常只使用前若干系数作为特征。

5) Formants（共振峰，F1、F2、F3）
- 含义：声音的共振峰，主要由声道形状决定。F1、F2、F3 对应不同的口腔开度和舌位，直接影响元音的质量。
- 应用：语音合成、语音识别、语言学研究（元音分类、音位识别）。
- 常见做法：通过估计声道的频率响应来提取共振峰，常用方法包括线性预测编码（LPC）和拟合的滤波器组。
- 解释：
  - F1 反映口腔的开度（较高的 F1 对应较张开的口腔）。
  - F2 反映舌位的前后位置（前位舌位通常带来较高的 F2）。
  - F3 提供额外的共鸣信息，帮助区分类似的元音音质。

快速对比与总结
- Pitches（基频）关注的是周期性振动和音高感知；Jitter 与 Shimmer 关注声带发声的稳定性与随机性（时间域的微小波动）。
- MFCCs 是频谱形状的统计描述，广泛用于识别任务的核心特征。
- Formants 是声道形状的物理实现，直接影响元音与音位的辨识。
- 实践中，常把 MFCCs 与 Formants、F1/F2 等结合起来，用于更鲁棒的声学表征。




## 和语言特征（词性、类符形符比、信息词、信息单元）。
# 1) Part-of-Speech (POS，词性)
- 含义
  - 词性标签把每个词归类到类别，如名词(N)、动词(V)、形容词(A)、副词(R) 等，反映句子的语法结构和语言风格。
- 常见用途
  - 研究语法复杂度、句法模式、信息载荷分布（content words vs function words）。
  - 区分信息载荷词（content words）与功能词（function words），帮助理解表达的信息密度与连贯性。
- 测量方式
  - 使用自然语言处理工具（如 spaCy、NLTK、Stanza）进行分词和POS 标注，统计各词性的频次与比例。
- 读解要点
  - 关注名词/动词/形容词的比例以及它们的连用模式，能揭示叙述的主题聚焦与信息表达方式。

# 2) Type-Token Ratio (TTR，类型-标记比)
- 含义
  - TTR = unique词数 / 总词数，用来衡量词汇多样性。Type 指不同的词（去重后的集合），Token 指文本中的词的总数。
- 常见用途
  - 评估语言产出中的词汇丰富度，帮助区分认知状态、语言流畅性等。
- 测量方式
  - 对文本分段后统计：TTR 越高，词汇多样性越丰富；TTR 也会随文本长度变化，需要在同等长度的片段内比较，或使用对长度更稳健的替代指标（如 moving-average TTR、Guiraud’s、-HD）。
- 读解要点
  - MCI 组可能呈现较低的TTR，表示词汇使用趋向重复、信息表达的多样性下降。

# 3) Information words（信息词）
- 含义
  - 信息词通常指承载语义核心的词，常包括名词、动词、形容词等“内容词”，用于传达事物、动作、属性等信息，而非语法功能的词。
- 常见用途
  - 衡量叙述的语义密度和信息载荷；信息词的比例常与叙述的清晰度和有效性相关。
- 测量方式
  - 在POS 标注基础上，筛选出 content words（如名词、实义动词、形容词、副词等）并统计数量或占比。
- 读解要点
  - 信息词比例下降可能意味着叙述信息密度下降，或叙述更偏向结构性、抽象性的表达。

# 4) Information units（信息单位）
- 含义
  - 信息单位是将叙述拆分为“独立信息点/信息块”的最小语义单位，常用于口语转写中评估信息传递的完整性和密度。一个信息单位通常对应一个事实、一个动作、一个关键属性等。
- 常见用途
  - 评价信息传递效率、叙事完整性，以及跨组的信息点数量差异（如 CN vs MCI）。
- 测量方式
  - 需要事先定义信息单位的划分准则（如基于语义成分、主谓宾结构、或基于人工标注的事实单元）。
  - 计算信息单位的总数、每句的信息单位数、信息单位密度（信息单位/句子或信息单位/词数）。
- 读解要点
  - 信息单位数量的减少或密度降低，提示叙述在信息传递上的有效性下降，可能与认知状态相关。


本研究使用的机器学习算法包括逻辑回归、随机森林（RF）、支持向量机（SVM）、高斯朴素贝叶斯（GNB）和k近邻（kNN）。

**结果：** 在训练中，语言特征的准确率通常高于声学特征。区分HC和AD的最高准确率为80.77%，由SVM实现，基于从语音数据中提取的所有特征；而区分HC与AD或MCI的最高准确率为80.43%，由RF实现，仅基于语言特征。

**结论：** 我们的结果表明语言特征（没提语音）在认知障碍自动诊断中的实用性和有效性，并验证了自动诊断在中文数据中的适用性。
