### 2️⃣ WER指标解释

**WER (Word Error Rate，词错误率)** 是语音识别最常用的评估指标：

```
WER = (插入错误 + 删除错误 + 替换错误) / 参考文本总词数

- 0.0 = 完美识别（0%错误）
- 0.2 = 优秀（20%错误）
- 0.5 = 可用（50%错误）
- 1.0 = 完全失败（100%错误）
```

**当前WER=0.9984的含义**：
- 模型几乎无法正确识别任何词语
- 预测结果与真实文本99.84%不匹配
- **训练失败，模型未学到有效特征**
